# Informativa sulla privacy di AppBanca

Ultimo aggiornamento: 17 luglio 2026

AppBanca è un'app di gestione delle finanze personali che funziona **interamente sul tuo dispositivo**.

## Quali dati vengono trattati

Quando importi un estratto conto (CSV o XLSX), AppBanca legge il file che scegli tu tramite il selettore di file del sistema e salva i movimenti, le categorie, le regole e i budget **solo nella memoria locale del tuo dispositivo** (nella cartella dati privata dell'app).

## Cosa NON facciamo

- Non raccogliamo, trasmettiamo o condividiamo nessun dato con noi o con terze parti.
- Non usiamo server, cloud o backend di alcun tipo.
- Non abbiamo strumenti di analisi, tracciamento o pubblicità.
- L'app non richiede né usa alcuna connessione a Internet su Android.

## Dove restano i tuoi dati

Tutti i dati (movimenti importati, categorie, regole, budget) restano nella memoria locale del tuo dispositivo, protetta dal sistema operativo Android come qualsiasi altra app. Disinstallando l'app, questi dati vengono eliminati insieme ad essa. Puoi inoltre eliminarli manualmente in qualsiasi momento dalle impostazioni dell'app.
